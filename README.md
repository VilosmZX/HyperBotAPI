# Disclaimer
This is a wa bot that is useful as an API for your frontend.
This API has cool features like:
* Auto reply
  * Add auto reply (/api/client/autoreply/add) 
    * Masukan data
     ```json
     {
       "trigger": "halo",
       "reply": "Hi"
     }
     ```

# Do this first
Install semua package
```
npm install
```
Migrate database
``` 
npm run migrate
```
Run production mode
```
npm run prod
```
Or
Run development mode
```
npm run dev
```
